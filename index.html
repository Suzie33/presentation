Types of TypeScript

TypeScript Data Types Syntax


						[Keyword] [Variable Name]: [Data Type] = [Value];
						[Keyword] [Variable Name]: [Data Type];

Example:


						let uname: string = "Suresh Dasari"; // string
						let age: number = 33; // number
						let isCompleted: boolean = true; // boolean
